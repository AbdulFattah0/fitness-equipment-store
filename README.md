#  Fitness Equipment Store – Full Stack Web Application

A full-stack fitness equipment e-commerce application built using **ASP.NET Core Web API** (backend) and **Vue.js with Quasar Framework** (frontend).

This project simulates a real online store where users can register, log in, browse products, manage a cart, place orders, and view their order history.

---

##  Tech Stack

### Backend (Server)
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- RESTful API
- DAO Pattern
- EF Migrations

### Frontend (Client)
- Vue 3
- Quasar Framework
- Vue Router
- Axios
- Responsive Mobile UI

---

## Features

- User Registration & Login
- JWT Authentication
- Browse Brands & Products
- Branch Locations
- Shopping Cart
- Place Orders
- Order History
- Secure Logout

---

##  Project Structure
/Casestudy → ASP.NET Core Web API (Backend)
